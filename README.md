## Installation

1.  Clone the repository :

```bash
   git clone https://github.com/SomnathRoy-JU25/Real-Time-Chat_App.git
```

2. Back-end Setup :

```bash
   cd server
   npm install express
```

3. Front-end Setup

```bash
   npm install
```

### Running the Application

1. Run the following command to start the application:

```bash
   npm run dev
```
